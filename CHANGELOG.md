# Changelog

All notable changes to **Create Ore Doubling** are documented in this file.

## [1.0.1] - 2026-02-16

### Documentation
- Updated README text for the CurseForge description and aligned it with the current feature set

### Added
- Configurable bonus drop chances for raw ore and raw ore block crushing (`rawOreCrushing.*` and `rawOreBlockCrushing.*`).
- Configurable chance for Experience Nuggets as a byproduct.

### Changed
- Rebalanced crushing recipes for raw ores and raw ore blocks.

## [1.0.0] - Initial Release

### Added
- Base mod with crushing recipes for Raw Iron, Raw Copper, Raw Gold, and Raw Zinc.
- Support for individual raw ores and raw ore blocks.
- German and English localization for configuration entries.
